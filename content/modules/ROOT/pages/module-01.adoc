= Module 1: Create Resources in Your Namespace
:imagesdir: ../assets/images

In this module you will create a ConfigMap and a Secret in your pre-assigned namespace.

== Exercise 1: Find your namespace in the OCP Console

. In the *OCP Console* tab, switch to the *Developer* perspective (top-left switcher).

. Click the *Project* dropdown and search for `zt-test`.
Select `zt-test-<your-username>`.

. Click *Config Maps* and *Secrets* in the left navigation — both are currently empty.

== Exercise 2: Create the ConfigMap and Secret

. Set your namespace variable:
+
[source,sh,role=execute]
----
export NS=zt-test-$(oc whoami)
echo "Using namespace: $NS"
----

. Create the ConfigMap:
+
[source,sh,role=execute]
----
oc create configmap zt-test-config \
  --from-literal=env=test \
  -n $NS
----

. Create the Secret:
+
[source,sh,role=execute]
----
oc create secret generic zt-test-secret \
  --from-literal=lab-key=zt-poc-value \
  -n $NS
----

== Verify

[source,sh,role=execute]
----
oc get configmap,secret -n $NS | grep zt-test
----

.Expected output
[source,texinfo]
----
configmap/zt-test-config   1      5s
secret/zt-test-secret      1      3s
----
