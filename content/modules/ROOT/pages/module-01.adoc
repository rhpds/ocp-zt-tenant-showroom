= Module 1: Create a ConfigMap
:imagesdir: ../assets/images

In this module you will create a ConfigMap in your pre-assigned namespace.

== Your access

[cols="1,2",options="header"]
|===
| Item | Value

| *Username*
| {user}

| *Password*
| {password}

| *Your namespace*
| {user}-zttest
|===

== Exercise 1: Log in to OCP and find your namespace

. In the *OCP Console* tab, log in with `{user}` / `{password}`.
. Switch to the *Developer* perspective (top-left switcher).
. Click the *Project* dropdown and select `{user}-zttest`.

== Exercise 2: Create the ConfigMap

. Log in to OCP from the command line:
+
[source,sh,role=execute]
----
oc login {openshift_console_url} -u {user} -p {password} --insecure-skip-tls-verify
----

. Set your namespace variable:
+
[source,sh,role=execute]
----
export NS=$(oc whoami)-zttest
echo "Using namespace: $NS"
----

. Create the ConfigMap:
+
[source,sh,role=execute]
----
oc create configmap zt-test-config \
  --from-literal=env=test \
  -n $NS
----

== Verify

[source,sh,role=execute]
----
oc get configmap zt-test-config -n $NS
----

.Expected output
[source,texinfo]
----
NAME              DATA   AGE
zt-test-config    1      5s
----
