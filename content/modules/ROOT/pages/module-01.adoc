= Module 1: Create a ConfigMap
:imagesdir: ../assets/images

In this module you will create a ConfigMap in your pre-assigned namespace.

== Exercise 1: Find your namespace

. In the *OCP Console* tab, switch to the *Developer* perspective.
. Click the *Project* dropdown and select `zt-test-<your-username>`.

== Exercise 2: Create the ConfigMap

. Set your namespace variable:
+
[source,sh,role=execute]
----
export NS=zt-test-$(oc whoami)
echo "Using namespace: $NS"
----

. Create the ConfigMap:
+
[source,sh,role=execute]
----
oc create configmap zt-test-config \
  --from-literal=env=test \
  -n $NS
----

== Verify

[source,sh,role=execute]
----
oc get configmap zt-test-config -n $NS
----

.Expected output
[source,texinfo]
----
NAME              DATA   AGE
zt-test-config    1      5s
----

Click *Next* to validate your work.
