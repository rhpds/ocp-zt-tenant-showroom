= Module 1: Create a ConfigMap
:imagesdir: ../assets/images

In this module you will create a ConfigMap in your first namespace `{user}-zttest`.

== Exercise: Create the ConfigMap

. Log in to OCP and set your namespace variable:
+
[source,sh,role=execute]
----
export NS={user}-zttest
echo "Working in namespace: $NS"
----

. Create the ConfigMap:
+
[source,sh,role=execute]
----
oc create configmap zt-test-config \
  --from-literal=env=test \
  -n $NS
----

=== Verify

[source,sh,role=execute]
----
oc get configmap zt-test-config -n $NS
----

.Expected output
[source,texinfo]
----
NAME              DATA   AGE
zt-test-config    1      5s
----

[TIP]
====
You can also view the ConfigMap in the *OCP Console* tab:
Developer perspective -> Config Maps -> `{user}-zttest`
====
