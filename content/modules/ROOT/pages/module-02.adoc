= Module 2: Label Your Namespace
:imagesdir: ../assets/images

In this module you will apply a label to your namespace to mark the lab as complete.

== Exercise 1: View current namespace labels in the OCP Console

. In the *OCP Console* tab, switch to the *Administrator* perspective.

. Go to *Home* -> *Projects* and click on `zt-test-<your-username>`.

. On the *Details* tab, find the *Labels* section.
Note that `lab-status` is not set yet.

== Exercise 2: Apply the label

. Set your namespace variable (if needed):
+
[source,sh,role=execute]
----
export NS=zt-test-$(oc whoami)
----

. Apply the label:
+
[source,sh,role=execute]
----
oc label namespace $NS lab-status=complete --overwrite
----

== Verify

[source,sh,role=execute]
----
oc get namespace $NS --show-labels | grep lab-status
----

.Expected output
[source,texinfo]
----
zt-test-devuser-poc   Active   ...   lab-status=complete,...
----

You can also refresh the OCP Console project details page — the `lab-status=complete` label will appear.
