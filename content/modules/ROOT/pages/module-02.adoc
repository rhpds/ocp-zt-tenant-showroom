= Module 2: Create a Secret in Your Second Namespace
:imagesdir: ../assets/images

In this module you will create a Secret in your second namespace `{user}-ztworkspace`.

[NOTE]
====
Many labs have multiple namespaces for different purposes. This module demonstrates working across two pre-assigned namespaces.
====

== Exercise: Create the Secret

. Set your second namespace variable:
+
[source,sh,role=execute]
----
export NS2={user}-ztworkspace
echo "Working in namespace: $NS2"
----

. Create the Secret:
+
[source,sh,role=execute]
----
oc create secret generic lab-status \
  --from-literal=status=complete \
  -n $NS2
----

=== Verify

[source,sh,role=execute]
----
oc get secret lab-status -n $NS2
----

.Expected output
[source,texinfo]
----
NAME         TYPE     DATA   AGE
lab-status   Opaque   1      5s
----
