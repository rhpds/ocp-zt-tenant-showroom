= Module 2: Create a Secret in Your Second Namespace

In this module create a Secret in `{user}-ztworkspace` — your second pre-created namespace.

== Your Task

[source,sh,role=execute]
----
oc create secret generic lab-status \
  --from-literal=status=complete \
  --from-literal=module=2 \
  -n {user}-ztworkspace
----

=== Verify

[source,sh,role=execute]
----
oc get secret lab-status -n {user}-ztworkspace
----

== Adding Dynamic Dependencies

If your playbooks need extra collections or Python packages, add to `runtime-automation/`:

[source]
----
runtime-automation/
├── requirements.txt    # pip install at container start
├── requirements.yml    # ansible-galaxy collection install at start
├── module-01/
└── module-02/
----

Example `requirements.yml`:
[source,yaml]
----
collections:
  - name: community.general
    version: ">=8.0"
----
