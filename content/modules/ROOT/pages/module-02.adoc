In this module you will create a `lab-status` Secret in your namespace.

|===
| Item | Value

| *Username*
| {user}

| *Your namespace*
| {user}-zttest
|===

== Exercise: Create the Secret

. Set your namespace variable:
+
[source,sh,role=execute]
----
export NS=$(oc whoami)-zttest
echo "Using namespace: $NS"
----

. Create the Secret:
+
[source,sh,role=execute]
----
oc create secret generic lab-status \
  --from-literal=status=complete \
  -n $NS
----

== Verify

[source,sh,role=execute]
----
oc get secret lab-status -n $NS
----

.Expected output
[source,texinfo]
----
NAME         TYPE     DATA   AGE
lab-status   Opaque   2      5s
----
