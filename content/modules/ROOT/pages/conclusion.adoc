= Developer Guide — Adding Solve/Validate to Your Lab

== Step 1: Add to AgV Catalog

[source,yaml]
----
# Required workload
workloads:
  - rhpds.ftl.ocp4_workload_runtime_automation_k8s

# Required showroom vars
ocp4_workload_showroom_runtime_automation_image: "quay.io/rhpds/zt-runner:v2.4.0"
ocp4_workload_showroom_wetty_image: "quay.io/rhpds/wetty:v3.0"
ocp4_workload_showroom_terminal_type: wetty
----

== Step 2: Create runtime-automation/ in Showroom Repo

[source]
----
runtime-automation/
├── requirements.txt        # optional: pip packages
├── requirements.yml        # optional: ansible collections
├── module-01/
│   ├── setup.yml           # optional pre-flight
│   ├── solve.yml           # creates resources
│   └── validation.yml      # checks with validation_check ✅/❌
└── module-02/
    └── ...
----

== Step 3: Write validation.yml (MANDATORY pattern)

[source,yaml]
----
---
- hosts: localhost
  connection: local
  gather_facts: false
  tasks:
    # Run your checks
    - kubernetes.core.k8s_info:
        kubeconfig: "{{ k8s_kubeconfig | default(omit) }}"
        validate_certs: false
        kind: ConfigMap
        name: my-config
        namespace: "{{ student_ns }}"
      register: r_cm

    # Build per-task booleans
    - ansible.builtin.set_fact:
        _task1_ok: "{{ r_cm.resources | length > 0 }}"

    # ALWAYS use validation_check — never ansible.builtin.fail
    - validation_check:
        check: "{{ _task1_ok }}"
        pass_msg: "✅ Task 1: ConfigMap exists"
        error_msg: "❌ Task 1: ConfigMap not found"
----

== Step 4: Test with curl

[source,bash]
----
# OCP labs (from laptop):
SHOWROOM=https://showroom-{guid}.apps....
curl -sk -X POST $SHOWROOM/runner/api/module-01/validation
curl -sk $SHOWROOM/runner/api/job/{Job_id}

# RHEL labs (from bastion):
curl -s -X POST http://localhost:8501/api/module-01/validation
curl -s http://localhost:8501/api/job/{Job_id}
----

== Common Mistakes

[cols="1,2",options="header"]
|===
| Mistake | Fix
| Using `ansible.builtin.fail` | Use `validation_check` — Output is empty with fail
| `ansible_user: student_user` on bastion | Use `bastion_user \| default('lab-user')`
| Wrong nookbag URL (`rhpds/nookbag`) | Must be `rhpds/nookbag-bundle`
| `setup-automation/` directory exists | Delete it — not needed with zerotouch/showroom chart
| `default-site.yml` instead of `site.yml` | Rename to `site.yml`
|===
